# Frameworks Overview\n\n## Flask\nFlask is a lightweight WSGI micro‑framework that provides the essentials for building web applications. It gives developers freedom to choose extensions and components, making it ideal for small to medium projects, prototyping, and services where fine‑grained control is desired.\n\n## Django\nDjango is a batteries‑included, high‑level framework that follows the "batteries‑included" philosophy, ORM, offering an­. It includes admin interface, authentication, templating, and many utilities, making it suited for full‑stack, data‑driven sites where consistency and built‑in features reduce boilerplate.\n\n## FastAPI\nFastAPI is a modern, async‑first framework built on Starlette and Pydantic. It leverages Python type hints to generate OpenAPI documentation automatically, delivering high performance comparable to Node or Go. It is optimal for high‑throughput APIs, micro‑services, and applications that benefit from async I/O and automatic interactive docs.
