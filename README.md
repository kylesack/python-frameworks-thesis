# Python Frameworks Thesis

This project involves research comparing three popular Python web frameworks: Flask, Django, and FastAPI. The goal is to evaluate their performance, ease of use, and suitability for various types of applications.

## Features
- **Flask**: Lightweight framework for building web applications.
- **Django**: High-level framework that encourages rapid development.
- **FastAPI**: Modern framework for building APIs with Python 3.6+ based on standard Python type hints.

## Technical Stack
- Python 3.x
- Flask, Django, FastAPI

## Usage
To run the project, clone the repository and install the required dependencies using:

```bash
pip install -r requirements.txt
```
